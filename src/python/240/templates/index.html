Music Store - REST API

Inputs

Select data to access/modify.

Directions

Use the following commands to access the API:

  • /json - receive a dump of all information on the server in JSON format.
  • /loop - prints a collection of the data collated using a loop in the template.
  • /getall - returns all the data on the server.
  • /get - returns the item matching the provided inputs.
  • /add - takes the user input and converts it to a new object to add to the server. Returns the new object if it was created successfully.
  • /update - takes the user input and updates the selected object's values, if it exists. Returns the new object if it was updated successfully.
  • /delete - deletes the selected object from the server.
